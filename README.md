# whitedove

A renderer of a specific superset of Markdown with focus in pure modality.

___

To install dependencies:

```bash
$ bun install
```

To run for development:

```bash
$ bun hot
```

___

- [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
